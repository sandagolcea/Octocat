git clone <git@github.com:userName/RepoName.git>
Copies the remote repository from github (bitbucket etc.) into your working area (i.e. working dir).
By default makes a directory to contain your repo, called <RepoName>.

$ git clone <git@github.com:userName/OldName.git> <NewName>
to clone in a directory called <NewName> instead of <OldName> (a by per default).
